# 📊 Business Intelligence & Automação de Comissões | Case Seazone

Este projeto apresenta a resolução de um desafio técnico focado em inteligência de vendas. O objetivo foi estruturar dados brutos de um CRM para automatizar o cálculo de comissões de Closers e SDRs.

## 🎯 Escopo do Projeto
A análise foi baseada em negócios do empreendimento "Novo Campeche Spot", com foco na padronização de dados e aplicação de regras financeiras para remuneração variável.

## 🛠️ Implementações Realizadas
- **Tratamento de Dados:** Limpeza de bases brutas e padronização de campos para garantir a integridade dos cálculos.
- **Modelagem Financeira:** Cálculo de valores brutos e líquidos (considerando 17,5% de impostos) e aplicação de multiplicadores conforme o canal de venda.
- **Governança de Dados:** Identificação de falhas no preenchimento do CRM, como negócios com valor zerado ou falta de justificativa de perda.
- **Visualização:** Estruturação de abas para consulta dinâmica por período (Mês/Ano).

## 📁 Organização
- `/data`: Bases originais de teste.
- `/results`: Relatórios e planilhas com os cálculos finais.
- `/docs`: Regras de negócio e premissas do desafio.
